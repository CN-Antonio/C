## 冒泡排序
n个元素，1&2比较，2&3比较，……，(n-1)&n比较（共n-1次）  

复杂度：n^2  
```c
//代码实现
int A[n],temp;
for(int i=0 ; i<n ; i++)
{
    for(int j=i ; j<n ; j++)
    {
        //
    }
}
```
